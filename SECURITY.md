## Reporting a vulnerability

Please **do not** open a public GitHub issue for security-sensitive reports.

Use one of:

- [GitHub Security Advisories](https://github.com/ArneCode/marser/security/advisories/new) (preferred), or
- Contact the maintainer via the repository owner profile: [ArneCode](https://github.com/ArneCode).

Include a description, affected versions, and steps to reproduce if possible. You should receive a response within a reasonable time; exact SLA is not guaranteed for this pre-1.0 project.
